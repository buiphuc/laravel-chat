# Changelog

All notable changes to `laravel-chat` will be documented in this file.

## v1.0.0 - 2026-03-25

### Added
- Multi-Actor System allowing polymorphic chat relationships.
- Capability-Based authorization for fine-grained permissions.
- In-memory Database testing setup.
- Auto-routing capabilities mapping incoming users to available admins.
- Real-time event broadcasting compatible with Laravel Reverb, Socket.IO, and Pusher.
- Block and Report Moderation APIs.
- Full-text search and extensive querying capabilities.
- Attachments upload system.
